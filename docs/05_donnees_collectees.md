# Données collectées — 5 startups sénégalaises réelles

## Structure de chaque startup

Chaque startup doit fournir les champs suivants :

| Champ | Type | Exemple | Obligatoire |
|---|---|---|---|
| **Nom** | Texte (1-100 car.) | Wave | Oui |
| **Secteur** | Énumération : `fintech`, `agritech`, `edtech`, `other` | Fintech | Oui |
| **Ville** | Texte (1-100 car.) | Dakar | Oui |
| **Année de création** | Entier (1900 - année courante) | 2018 | Oui |
| **Incubée** | Booléen (oui/non) | Non | Oui |
| **Description** | Texte (0-500 car.) | Application de transfert d'argent... | Non |
